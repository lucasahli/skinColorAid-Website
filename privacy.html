Privacy Policy

Last Updated: December 15, 2025

1. Data Minimization

SkinColorAid is built on the principle of absolute privacy. We do not collect, store, share, or sell your personal data. The app functions as a standalone tool on your machine.

2. Local Processing Only

The application analyzes color values from your screen buffer. This processing happens entirely within your device's Random Access Memory (RAM).

  • No data is sent to the cloud.
  • No analytics or tracking scripts are embedded in the app.
  • Screen data is discarded the millisecond the analysis frame is rendered.

3. macOS Permissions

The app requires Screen Recording permissions to sample pixels. This is a technical requirement of the macOS operating system for any tool that "sees" what is on screen (like color pickers). We use this strictly to calculate color vectors.
